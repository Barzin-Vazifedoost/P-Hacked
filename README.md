# P-Hacked

A modern React dashboard that simulates repeated null-hypothesis experiments and visualizes the resulting p-value distribution.

## Features

- Controls for simulation count and per-group sample size
- Repeated two-sample significance testing under the null hypothesis
- Recharts histogram with a highlighted `p = 0.05` threshold
- Insights panel for false-positive rate and run summary
- Tailwind-powered dark dashboard layout

## Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run lint
npm run build
```
